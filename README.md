# Tic Tac Toe Game 🎮

A simple and responsive **Tic Tac Toe game** built using **HTML, CSS, and JavaScript**. This project demonstrates basic game logic, DOM manipulation, and responsive web design so the game works smoothly on both desktop and mobile devices.

## 🚀 Features

* Classic **2-player Tic Tac Toe gameplay**
* **Responsive design** (works on mobile, tablet, and desktop)
* **Interactive UI** with smooth user experience
* **Win detection logic**
* **Draw detection**
* **Game reset functionality**

## 🛠️ Built With

* **HTML** – Structure of the game board and layout
* **CSS** – Styling and responsive design
* **JavaScript** – Game logic, event handling, and win checking

## 🎯 How to Play

1. Player **X** starts the game.
2. Players take turns clicking on empty cells.
3. The first player to get **three marks in a row** (horizontal, vertical, or diagonal) wins.
4. If all cells are filled and no player wins, the game ends in a **draw**.
5. Click the **Reset / Restart button** to start a new game.

## 📱 Responsiveness

The game layout automatically adjusts to different screen sizes using responsive CSS, ensuring a smooth experience on:

* Mobile devices
* Tablets
* Desktop screens
